# Changelog

## [0.2.0] - 2026-07-21

### Added

- “看—测—想—验—练—定”六步预习法。
- 每单元两道前置小测。
- 苏格拉底式分步引导、最多三级提示和自我解释清单。
- 基于错误选项映射的动态错因诊断。
- 母题、换数字、换问法、换情境的一题三变体系。
- “已理解 / 仍需关注 / 正式上课重点”预习成果单。
- 动态打印知识卡、错因卡、变式练习和完整成果单。
- schema_version 2.0、六步校验规则和新版示例。

### Changed

- 第六步由“带回家”改为“定重点”，打印降级为成果保存功能。
- 单元网页改为六步导航和分屏式学习路径。
- 网站测试新增六步容器、打印包和 JavaScript 语法检查。

## [0.1.0] - 2026-07-19

- 初始版本：知识卡、互动例题、三关挑战和学期总览。
